        Virtual private network is a network architecture for virtually extending a private network across one or multiple other networks which are either untrusted or need to be isolated. A VPN can extend access to a private network to users who do not have direct access to it, such as an office network allowing secure access from off-site over the Internet. This is achieved by creating a link between computing devices and computer networks by the use of network tunneling protocols.
